Lab 14

Цель работы
Освоить модульное тестирование (Jest) и отладку в DevTools:
breakpoints, step-over/into, watchers.

Задачи
- Реализовать 2 функции.
- Написать модульные тесты (один тест должен падать из-за ошибки).
- Найти и исправить ошибку в DevTools:
  - установить breakpoint;
  - выполнить шаги (Step Over / Into);
  - проверить значения переменных (Watch).



Описание функций
1. arraySum(arr)
Возвращает сумму всех элементов массива.

2. findMax(arr)
Должна искать максимальный элемент, но содержит *специальную ошибку*  
в строке сравнения num < max.
